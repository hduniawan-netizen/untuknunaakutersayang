# Valentine's Day Interactive Page

## Overview
This project is an interactive web page designed to celebrate Valentine's Day. It offers users a unique and engaging experience, incorporating themed graphics, animations, and activities.

## Features
- **Interactive Elements**: Users can click on various elements to reveal surprises.
- **Animations**: Smooth animations that enhance the visual appeal.
- **User Interactivity**: Users can submit their messages or wishes.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hduniawan-netizen/untuknunaakutersayang.git
   ```
2. Navigate to the project directory:
   ```bash
   cd untuknunaakutersayang
   ```
3. Open `index.html` in a web browser to see the interactive page.

## Usage
- Simply access the page in a web browser and engage with the interactive elements.

## Technologies Used
- HTML, CSS, and JavaScript for front-end development.
- Libraries such as jQuery for animations.
